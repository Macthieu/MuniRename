# Contribuer a MuniRenommage

Merci de contribuer a MuniRenommage.

## Principes

- MuniRenommage reste un outil specialise de renommage; Orchiviste reste le hub.
- Priorite a la fiabilite, la lisibilite et la coherence documentaire.
- Pas de refactor metier majeur sans decision explicite.

## Workflow

- Branche de travail: `feature/*`, `fix/*`, `docs/*`, `chore/*`.
- Commits atomiques et messages explicites.
- Pull Request avec description des impacts fonctionnels et documentaires.

## Versionnage

- Semantic Versioning obligatoire.
- Mettre a jour `CHANGELOG.md` pour les changements notables.

## Licence

En contribuant, vous acceptez une publication sous GNU GPL v3.0.

## Note de transition

Le nom public est `MuniRenommage`. Certains identifiants techniques internes peuvent encore utiliser `MuniRename` temporairement.
